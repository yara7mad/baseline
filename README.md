# Baseline: AI-Powered PV Design Review Tool

**Baseline** is an automated infrastructure auditing system designed to streamline the review of Photovoltaic (PV) solar designs. 

Developed by **Yara Abu Ghazalah**, for Desert Technologies the company
This project was developed for the **Futurethon** hackathon, where it was recognized as a finalist project. Following the demonstration, Desert Technologies—a leader in Saudi Arabia's renewable energy sector—invited me to collaborate and further develop this solution for their engineering workflows.

## Overview
As Saudi Arabia accelerates toward its Vision 2030 sustainability goals, the demand for rapid, accurate renewable energy infrastructure auditing has never been higher. **Baseline** addresses this by using **Llama-3.3-70B** (via Groq API) to automate the extraction of technical specifications from complex engineering PDFs.

## 🛠️ Tech Stack
* **Backend:** Node.js, Express.js
* **AI Engine:** Groq API (Llama-3.3-70B-Versatile)
* **Parsing:** PDF-Parse for deep technical data extraction
* **Frontend:** Interactive Engineering Dashboard with real-time NCR generation
